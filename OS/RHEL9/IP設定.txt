RockyLinux9 のIP設定

◆OSバージョン確認
# cat /etc/redhat-release
Rocky Linux release 9.2 (Blue Onyx)
#

◆NetworkManager 接続プロファイルリストの確認
nmcli connection show

NAME    UUID                                  TYPE      DEVICE
ens192  XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX  ethernet  ens192
lo      YYYYYYYY-YYYY-YYYY-XXXX-YYYYYYYYYYYY  loopback  

◆IP設定
例）nmcli connection modify ens192 ipv4.method manual ipv4.address 192.168.1.10/24 ipv4.gateway 192.168.1.1 ipv4.dns 192.168.1.1 ipv4.dns-search example.com

※ipv4.method manualというのはDHCPを使用しないという意味です。
